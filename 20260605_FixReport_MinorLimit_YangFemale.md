# 修正報告：陽女小限起始地支錯誤

**日期：** 2026-06-05  
**Commit：** `7075fc9`  
**檔案：** `chart-api.js`  
**修正者：** Raziel（Claude Sonnet 4.6）

---

## 一、問題描述

`calcFlowYearAges()` 函數將「陽女」錯誤歸為**順行**（與陰男同組），
且未設定獨立的陽女起始地支，導致陽女小限走錯宮位、方向完全相反。

### 錯誤根因

```js
// 修正前（錯誤）
const isForward  = yinYang === '陰男' || yinYang === '陽女';  // 陽女被歸入順行
const startBranch = isForward ? config[0] : config[1];        // 無陽女獨立起點
```

正確規則：
- **陰男** → 順行（+1）
- **陽男 / 陰女 / 陽女** → 逆行（-1）
- **陽女**起始地支全五行局統一為 **戌**

---

## 二、修正內容

### 2-1 `MINOR_LIMIT_START` 常數結構調整

config 從二元素改為**三元素**：  
`[陰男起點（順行）, 陽男陰女起點（逆行）, 陽女起點（逆行）]`

```js
// 修正前
const MINOR_LIMIT_START = {
  '水二局': ['丑', '申'],
  '木三局': ['巳', '申'],
  '金四局': ['丑', '申'],
  '土五局': ['未', '辰'],
  '火六局': ['申', '寅'],
};

// 修正後
const MINOR_LIMIT_START = {
  '水二局': ['丑', '申', '戌'],
  '木三局': ['巳', '申', '戌'],
  '金四局': ['丑', '申', '戌'],
  '土五局': ['未', '辰', '戌'],
  '火六局': ['申', '寅', '戌'],
};
```

### 2-2 `calcFlowYearAges()` 方向邏輯修正

```js
// 修正前
const isForward  = yinYang === '陰男' || yinYang === '陽女';
const startBranch = isForward ? config[0] : config[1];

// 修正後
const isForward  = yinYang === '陰男';   // 只有陰男順行
const startBranch = yinYang === '陽女' ? config[2]   // 陽女獨立起點
                  : isForward           ? config[0]   // 陰男
                  :                       config[1];  // 陽男/陰女
```
